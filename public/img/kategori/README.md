# Folder Gambar Kategori

Folder ini berisi gambar-gambar untuk kategori billiard.

## Gambar yang Diperlukan:

1. **8ball.webp** - Gambar untuk kategori 8 Ball
   - Ukuran: 200x150px atau lebih
   - Format: WebP, JPG, atau PNG

2. **9ball.jpg** - Gambar untuk kategori 9 Ball  
   - Ukuran: 200x150px atau lebih
   - Format: JPG atau PNG

3. **vip.jpg** - Gambar untuk kategori Meja VIP
   - Ukuran: 200x150px atau lebih
   - Format: JPG atau PNG

4. **snooker.webp** - Gambar untuk kategori Snooker
   - Ukuran: 200x150px atau lebih
   - Format: WebP, JPG, atau PNG

## Cara Menambahkan Gambar:

1. Siapkan gambar dengan nama file yang sesuai
2. Copy gambar ke folder `public/img/kategori/`
3. Pastikan nama file sesuai dengan yang ada di kode

## Contoh Gambar yang Bisa Digunakan:

- 8 Ball: Foto bola billiard nomor 8 (hitam)
- 9 Ball: Foto bola billiard nomor 9 (kuning bergaris)
- VIP: Foto meja billiard mewah atau ruang VIP
- Snooker: Foto meja snooker dengan bola-bola snooker
